# 🧠 One-Poem-GPT （一首诗 GPT）

这是一个极简的 GPT 模型，只用 **一首唐诗《静夜思》** 作为训练数据，演示 **过拟合现象**。  
通过实验可以看到：当数据量过小时，模型会“死记硬背”，几乎只能复述原诗。

---

## ✨ 实验目标
- 理解 **GPT（自回归语言模型）** 的基本原理  
- 学会用 **Transformer from scratch 极简实现** 搭建模型  
- 演示 **过拟合现象**：小数据集 → 模型记忆，而不是泛化
